# Proyecto-JAVA-Punto-de-Venta-con-Usuarios-y-M-todos-de-Pago-
Sistema de Punto de Venta desarrollado en Java con interfaz gráfica (Swing). Incluye inicio de sesión por roles (Administrador y Cliente), gestión de inventario, carrito de compras y múltiples métodos de pago. Ideal para prácticas de programación orientada a objetos.
