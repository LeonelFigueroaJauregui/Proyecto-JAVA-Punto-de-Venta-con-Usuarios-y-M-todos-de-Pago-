# Images Directory

This folder is used to store images and icons for the POS System user interface.

## Recommended Image Assets

- `logo.png` - Application logo
- `user-icon.png` - User avatar icon
- `cart-icon.png` - Shopping cart icon
- `product-placeholder.png` - Default product image
- `payment-icons/` - Payment method icons (cash, card, transfer)

## Image Specifications

- Recommended format: PNG with transparency
- Logo size: 200x200 pixels
- Icons size: 32x32 or 64x64 pixels
- Product images: 150x150 pixels
